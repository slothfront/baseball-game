# ⚾ baseball-game — 2人対戦ベースボール（ボール隠し読み合いルール）

同じ端末・同じキーボードで2人で遊べる、心理戦の野球ゲームです。
ブラウザだけで動く単一HTMLファイル（依存ライブラリなし）。

## あそびかた

`index.html` をブラウザで開くだけ。プレイヤー名とイニング数（3回／5回）を決めてスタート。

1. **守備側**が、`1塁打` `2塁打` `3塁打` `ホームラン` の **どこか1つにボールを隠します**
   （攻撃側は画面を見ないでください 🙈）。
2. **攻撃側**が4択から1つを選びます。
   - **当たり** → その塁ぶん進塁！ 打者が交代し、守備がもう一度隠し直します。
   - **ハズレ** → **1アウト**。その塁は封鎖され、残りから選び直し（ボールは同じ場所のまま）。
3. **3アウト**で攻守交代。両チームが攻撃して1イニング。最終回終了時に得点が多い方が勝ち。
   （同点なら延長／最終回裏で逆転すればサヨナラ勝ち）

## 操作

- マウス / タップでカードを選択、またはキーボードの `1` `2` `3` `4`。

## 開発メモ

- 単一ファイル `index.html`（HTML + CSS + Vanilla JS）。ビルド不要。
- ローカル2人対戦専用（オンライン対戦・AIなし）。
